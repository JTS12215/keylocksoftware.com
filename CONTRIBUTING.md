# Contributing to KeyLock

We welcome contributions! To keep the project secure and organized:
1. **Security First:** If you find a vulnerability, please do not open a public issue. Email keylock.app@gmail.com instead.
2. **License Agreement:** By contributing, you agree that your contributions will be licensed under the project's GPLv3 license.
3. **Coding Standards:** Please follow the style guides provided in the `/docs` folder.
4. **Pull Requests:** Ensure your PR description clearly explains the change and includes tests where applicable.
